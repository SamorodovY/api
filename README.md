# Тестирование API
Провел полное тестирование REST API публичного сервиса Petstore (Swagger). Параллельно создал рабочую SOAP-коллекцию для сервиса Country Info, позволяющую эффективно получать и проверять информацию о валюте, языке и других атрибутах различных стран.
